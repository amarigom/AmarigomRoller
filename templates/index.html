{% if lang == 'es' %}Como Tomar Medidas{% else %}How to Measure{% endif %}

{% if lang == 'es' %}Tips importantes antes de solicitar tu cotizacion{% else %}Important tips before requesting your quote{% endif %}

{% if lang == 'es' %}Cortinas Roller{% else %}Roller Blinds{% endif %}

  • 1
    {% if lang == 'es' %}Ancho:{% else %}Width:{% endif %} {% if lang == 'es' %} Medir el ancho del vano de la ventana y sumar 5 cm de cada lado (10 cm en total) si hay espacio disponible. Esto evita filtraciones de luz por los costados. {% else %} Measure the window opening width and add 5 cm on each side (10 cm total) if space is available. This prevents light leakage from the sides. {% endif %}
  • 2
    {% if lang == 'es' %}Alto:{% else %}Height:{% endif %} {% if lang == 'es' %} Medir desde donde se colocara el soporte hasta el largo deseado. Tener en cuenta que se necesitan aproximadamente 30 cm adicionales de tela que quedan enrollados en el tubo superior. {% else %} Measure from where the bracket will be placed to the desired length. Keep in mind that approximately 30 cm of additional fabric stays rolled on the upper tube. {% endif %}
  • 3
    {% if lang == 'es' %}Tipo de ventana:{% else %}Window type:{% endif %} {% if lang == 'es' %} Verificar si la ventana esta en un angulo, empotrada en la pared o tiene salientes. Esto afecta la instalacion y el tipo de soporte necesario. {% else %} Check if the window is at an angle, recessed into the wall, or has protrusions. This affects the installation and the type of bracket needed. {% endif %}
  • 4
    {% if lang == 'es' %}Espacio libre:{% else %}Clear space:{% endif %} {% if lang == 'es' %} Comprobar que no haya manijas, cerraduras u otros elementos que interfieran con el movimiento de la cortina al subir y bajar. {% else %} Make sure there are no handles, locks or other elements that interfere with the curtain movement when raising and lowering. {% endif %}

{% if lang == 'es' %}Cortinas Tradicionales{% else %}Traditional Curtains{% endif %}

  • 1
    {% if lang == 'es' %}Alto total:{% else %}Total height:{% endif %} {% if lang == 'es' %} Medir desde el zocalo (o moldura superior) hasta la posicion exacta donde se colocara el barral. Si el barral ya esta instalado, medir desde el barral hasta el piso o largo deseado. {% else %} Measure from the baseboard (or upper molding) to the exact position where the rod will be placed. If the rod is already installed, measure from the rod to the floor or desired length. {% endif %}
  • 2
    {% if lang == 'es' %}Ancho del barral:{% else %}Rod width:{% endif %} {% if lang == 'es' %} El barral debe sobresalir entre 15 y 20 cm de cada lado de la ventana para permitir que la cortina se abra completamente y no bloquee la luz natural. {% else %} The rod should extend 15 to 20 cm beyond each side of the window to allow the curtain to open fully and not block natural light. {% endif %}
  • 3
    {% if lang == 'es' %}Caida de la tela:{% else %}Fabric drape:{% endif %} {% if lang == 'es' %} Para un efecto elegante con pliegues, el ancho de tela recomendado es entre 1.5x y 2x el ancho de la ventana. Mas tela genera pliegues mas ricos y un aspecto mas lujoso. {% else %} For an elegant pleated effect, the recommended fabric width is between 1.5x and 2x the window width. More fabric creates richer folds and a more luxurious look. {% endif %}
  • 4
    {% if lang == 'es' %}Distancia al piso:{% else %}Distance to floor:{% endif %} {% if lang == 'es' %} Decidir si la cortina llegara al piso (dejar 1-2 cm de separacion) o si tendra un largo medio. Las cortinas que tocan el piso dan un aspecto mas formal y elegante. {% else %} Decide if the curtain will reach the floor (leave 1-2 cm clearance) or have a mid-length. Floor-length curtains give a more formal and elegant appearance. {% endif %}
